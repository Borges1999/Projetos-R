# 📊 Bellabeat Case Study – Data Analysis in R

## 📌 Sobre o Projeto

Este projeto apresenta uma análise completa de dados comportamentais de usuários de dispositivos inteligentes da Bellabeat, com foco na geração de insights estratégicos para apoiar decisões de marketing e crescimento do negócio.

A análise foi desenvolvida em **R**, aplicando uma pipeline de dados ponta a ponta — desde a importação e limpeza até a geração de recomendações executivas.

---

## 🎯 Objetivo de Negócio

Responder questões estratégicas como:

- Quais são algumas tendências no uso de dispositivos inteligentes?
- Como essas tendências poderiam se aplicar aos clientes da Bellabeat?
- Como essas tendências poderiam ajudar a influenciar a estratégia de marketing da Bellabeat?

---

## 🔎 Etapas do Projeto

- Entendimento do problema de negócio
- Importação e limpeza dos dados
- Tratamento e padronização das variáveis
- Análises exploratórias (EDA)
- Correlações e identificação de padrões
- Criação de visualizações estratégicas
- Geração de recomendações orientadas a dados

## 💡 Diferenciais do Projeto

- Pipeline estruturada ponta a ponta
- Comunicação orientada a stakeholders
- Conexão entre análise técnica e estratégia de negócio
- Organização profissional de arquivos

---

- **Fonte:** [[(https://www.kaggle.com/datasets/sujalsuthar/amazon-delivery-dataset)](https://www.kaggle.com/datasets/sujalsuthar/amazon-delivery-dataset)](https://www.kaggle.com/datasets/arashnic/fitbit)

## 🚀 COMO REPRODUZIR AS ANÁLISES

Siga os passos abaixo para executar o projeto em seu ambiente local:

1. **Baixe ou clone o repositório** para o seu computador:
   - Se for baixar: clique em `Code` > `Download ZIP` e extraia os arquivos.
   - Se for clonar via Git:
     ```bash
     git clone https://github.com/seu-usuario/seu-repositorio.git
     ```
     
2. **Abrir no RStudio**

- Abra o **RStudio**
- Vá em **File > Open File**
- Selecione os arquivos **bellabeat_script.R** e **projeto_bellabeat.R**

3. **Executar as análises**
   
- Execute as etapas na ordem apresentada no projeto:
  1. Pacotes
  2. Importação
  3. Tratamento dos dados
  4. Análise Exploratória
  5. Análise Estatística Descritiva
  6. Métricas
  7. Visualizações

**Obs.:** Caso não consiga reproduzir, talvez precise setar de forma manual o caminho da pasta onde os arquivos se encontram, para isso use a função set("caminho_dos_arquivos") no console R.
